# 基础入门

## 正交叉

![](./imgs/正交叉1.jpg)

正交叉重心在两脚中间，脚要保持一前一后，重心没有明显上下起伏，通过外八、内八、交叉向前移动

![](./imgs/正交叉2.jpg)

这个时候就需要脚主动改变方向之后在发力向两侧滑行

![](./imgs/正交叉3.jpg)

向前移动的时候重心需要一直保持在桩线上面，不可离开桩线

![](./imgs/正交叉4.jpg)

这里是重心
